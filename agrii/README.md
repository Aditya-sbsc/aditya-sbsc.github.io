# Indonesia Fertilizer Website

A modern, responsive website for Indonesia Fertilizer company showcasing their organic fertilizer products. 

## Features

- ✅ Fully responsive design
- ✅ Smooth scrolling navigation
- ✅ Parallax effects
- ✅ Interactive animations
- ✅ Mobile-friendly menu
- ✅ Scroll-to-top button
- ✅ SEO-friendly structure

## Required Images

Place these images in the same directory as your HTML file:

1. **logo.png** - Company logo
2. **farmer.png** - Farmer illustration
3. **barn.png** - Barn illustration
4. **leaves-left.png** - Left decorative leaves
5. **leaves-right.png** - Right decorative leaves
6. **fertilizer-bag.png** - Fertilizer bag product image
7. **fertilizer-bowl.png** - Bowl with fertilizer
8. **wheat.png** - Wheat stalks illustration
9. **plant-pot.png** - Plant pot icon
10. **hand-plant.png** - Hand holding plant
11. **strawberry.jpg** - Strawberry plant photo
12. **lawn.jpg** - Green lawn photo
13. **soil.jpg** - Organic soil photo

## Installation

1. Download all files (index.html, styles.css, script.js)
2. Place all required images in the same folder
3. Open `index.html` in your web browser

## Customization

### Colors
Main colors used:
- Primary: `#4a9d9f` (Teal)
- Secondary: `#2c5f6f` (Dark Teal)
- Sky: `#87CEEB` (Light Blue)
- Green: `#90EE90` (Light Green)

### Fonts
The website uses Arial as the primary font. You can change it in the CSS:

```css
body {
    font-family: 'Your Font', sans-serif;
}
```

### Content
Edit the text content directly in the HTML file.  All text is in Indonesian and English.
